# willow64-doc

## Willow64キーボードについて

Willow64は手指の自然な動きに優しくフィットするキー配列が特徴の、左右分離型の自作キーボードです。


Willow64の紹介記事 - [自作キーボード Willow 64を作った話](https://coal.hatenablog.com/entry/2020/12/04/231923)
